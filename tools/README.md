# Port Scanner

## Languages

- C
- Python

## Structure

- c/ → C implementation
- python/ → Python implementations

## Usage

### C

```bash
gcc port_scanner.c -o scanner
./scanner
```
### Python

```bash
python3 port-scanner-v1.0.py 
python3 port-scanner-v1.2.py 
python3 port-scanner-v1.3.py
```
